# Build to Launch: Newsletter Engine

> Premium plugin — [Get access on Gumroad](https://buildtolaunch.gumroad.com/l/newsletter-engine)

Full newsletter production system for AI builders. Goes from research to published Substack post in one workflow.

## What's included

| Skill | What it does |
|---|---|
| `/research-digest` | Pull weekly research across your topic — Reddit, Hacker News, web |
| `/article-draft` | Draft a full article from your notes or research digest |
| `/notes-generate` | Generate 3–5 Substack notes from a published article |
| `/seo-check` | Validate title, description, and structure for search |
| `/publish-substack` | Push draft to Substack via API |

## How to install

1. Purchase on [Gumroad](https://buildtolaunch.gumroad.com/l/newsletter-engine)
2. Download the plugin files from your Gumroad library
3. Add to your Claude Code plugins directory
4. Run `/article-draft` to start your first piece

## Who this is for

Newsletter writers publishing 1–3x per week who want to cut production time in half without losing their voice.
