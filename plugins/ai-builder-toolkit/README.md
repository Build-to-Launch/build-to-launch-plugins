# Build to Launch: AI Builder Toolkit

> Premium plugin — [Get access on Gumroad](https://buildtolaunch.gumroad.com/l/ai-builder-toolkit)

Content system built specifically for AI builders publishing in public. Mon/Wed/Fri cadence, content pillars, and cross-platform distribution.

## What's included

| Skill | What it does |
|---|---|
| `/weekly-plan` | Generate your 3-post weekly content plan from your current build |
| `/build-in-public` | Turn a build update or lesson into a Monday post |
| `/content-pillars` | Map any idea to the right content pillar and format |
| `/distribute` | Push finished content to Substack, Dev.to, Reddit, and Bluesky |

## How to install

1. Purchase on [Gumroad](https://buildtolaunch.gumroad.com/l/ai-builder-toolkit)
2. Download the plugin files from your Gumroad library
3. Add to your Claude Code plugins directory
4. Run `/weekly-plan` to generate your first week

## Who this is for

AI builders who are building products in public and want a repeatable content system — without hiring a content team.
